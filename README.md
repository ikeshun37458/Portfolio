# 家計簿予測Webアプリ

過去の家計簿のデータから来月の支出を予測するWebアプリです。

## 主な機能

・家計簿データのアップロードと分析  
・毎月の支出をカテゴリ別に可視化  
・カテゴリ別で直近12ヶ月のデータから来月の支出を予測  

## 使用技術

・フロントエンド： Streamlit  
・バックエンド： FastAPI  
・機械学習モデル： TensorFlow  
・その他： Pandas, Matplotlib
